# Forms Workshop

A demonstration pizza ordering form showcasing accessible form design patterns.

## Overview

Workshop example demonstrating various aspects of accessible form design, including proper labeling, grouping, validation, and error handling.

## Features

- Proper semantic HTML form structure
- Form field labeling and association
- Fieldset and legend grouping for radio buttons
- ARIA attributes for enhanced accessibility
- Client-side validation with error messaging
- Accessible error announcements using ARIA live regions

## Usage

Open `index.html` in a web browser to view and interact with the demo form. The form includes pizza customization, delivery information, and payment fields.

## Technologies

- HTML5
- CSS3
- jQuery 3.6.0
